To locate the java command on your system, do the following:

- If you're using the version of Java that IntelliJ automatically installs (a derivative of OpenJDK)
-- IntelliJ displays the full path of the Java command at the beginning of the Run window when you run the program.
-- You can use the path displayed in the Run window to access java from the command line

- If you're using the Oracle version of Java
-- See the following URL for the installation location on your platform
-- https://docs.oracle.com/en/java/javase/17/install
