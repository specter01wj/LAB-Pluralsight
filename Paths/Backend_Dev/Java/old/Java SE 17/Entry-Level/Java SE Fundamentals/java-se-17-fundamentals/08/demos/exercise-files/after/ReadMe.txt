The after version of the MathEquation class has been updated to have all of the fields marked as private and to have accessors/modifiers(getters/setters) for each field.

This was done using the same techniques shown in the slides for the fields within the Flight class.

Be sure to clear out any program arguments you've added to the IntelliJ Run Configuration
- Within IntelliJ, choose the "Run" menu
- Select "Edit Configurations..."
- Remove any conntents from the "Program arguments" field
- Click "OK"
