# Installing Node

Visit https://nodejs.org/en/download and choose the your operating system, version manager, and package manager.

# Installing TypeScript/tsc

After Node is installed, type `npm install -g typescript` from a Terminal window.

# Use tsc to Compile TypeScript to JavaScript

In Terminal in the directory with a `.ts` file, type `tsc <FILENAME.ts>`.

For example, if your file is named `tax.ts`, type `tsc tax.ts` to compile to a JavaScript file.
